\subsection{列数が少ないテーブル（8列以下）}

通常の adjustbox を使用:

\toprule

\end{adjustbox}

\end{table}

\section{修正の効果}

修正後の特徴:

\begin{itemize}
\item 列数 > 8 のテーブル: {\small フォントサイズを削減
\item adjustbox: max height 制限で縦方向をページ内に収束
\item max height=0.85\textbackslash{}textheight: ページの85\%までを使用可
\end{itemize}
